# 命令文
クイズを作っています。
あまり知られていない興味深い豆知識やトリビアに関するクイズを作ってください。

# 欲しい回答
* 問題(str)
* 選択肢3つ(List[str])
* 回答番号(int)
* クイズの対する解説(str)

# 出力フォーマット (JSON)
{{
    "item": [
        {{
            "quiz": "問題文",
            "choices": ["選択肢1", "選択肢2", "選択肢3"],
            "answer": 0,  // 正解の選択肢（0-2）
            "explanation": "解説"
        }},
        ...
    ]
}}

# クイズのカテゴリー
以下のカテゴリーについてのクイズでお願いします。
「{category}」
